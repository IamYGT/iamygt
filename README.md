## 👋 About Me

I'm **Ercan**, a tech entrepreneur and full-stack developer based in **Tirana, Albania** 🇦🇱.
I started coding at **13**, won a **Google Developer Scholarship** in 2017, and have been building real products ever since.

- 🏢 **CEO @ [YGT Labs](https://ygtlabs.com)** — software company focused on mobile & web
- ⚙️ **Co-Founder @ [Memsidea](https://memsidea.com)** — digital agency (est. 2016)
- 🚗 **Creator of ProEcuPortal & KingEcu** — leading ECU tuning software platforms
- 🌍 **Building [ecutuningportal.com](https://ecutuningportal.com)** — multilingual B2B SaaS (24 languages)
- 🤖 **[ygtwa](https://github.com/IamYGT/ygtwa)** — self-hosted WhatsApp Web REST API & MCP bridge in Go
- 🏎️ Passionate about **Formula 1**, motorsports & exploring new cultures
- 📍 **Tirana, Albania** (CET/UTC+1 — remote-friendly worldwide)
